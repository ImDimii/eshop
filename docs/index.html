Installazione

Requisiti di Sistema

  • PHP 7.4 o superiore
  • MySQL 5.7 o superiore
  • Apache 2.4 o superiore

Moduli PHP Richiesti

  • php-mysql (PDO e MySQLi)
  • php-gd (Gestione immagini)
  • php-curl (API pagamenti)
  • php-zip (Gestione archivi)
  • php-mbstring (Supporto UTF-8)
  • php-xml (Gestione XML)

Windows

1. Installare XAMPP

  • Scaricare XAMPP da apachefriends.org
  • Eseguire installer come amministratore
  • Selezionare componenti:
    • Apache
    • MySQL
    • PHP
    • phpMyAdmin

2. Installare Composer

  • Scaricare da getcomposer.org
  • Durante l'installazione selezionare PHP di XAMPP
  • Verificare l'installazione:
composer --version

3. Configurare Database

  • Aprire phpMyAdmin (http://localhost/phpmyadmin)
  • Creare nuovo database 'eshop'
  • Importare database.sql
CREATE DATABASE eshop CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

4. Configurare Progetto

cd C:\xampp\htdocs // Modificare config.php: define('DB_HOST', 'localhost'); define('DB_USER', 'root'); define('DB_PASS', ''); define('DB_NAME', 'eshop');

5. Impostare Permessi

  • Aprire Esplora File
  • Per ogni cartella:
    • uploads/products/
    • uploads/receipts/
    • logs/
    • temp/
  • Propriet√† ‚Üí Sicurezza ‚Üí Modifica
  • Aggiungere permessi completi per utente IUSR

6. Accesso Pannello Admin

  • URL: http://localhost/admin/
  • Credenziali predefinite:
    • Username: admin
    • Password: admin123
IMPORTANTE: Cambiare immediatamente la password al primo accesso!

Linux

1. Installare LAMP Stack

sudo apt update sudo apt install apache2 mysql-server # Installare PHP e moduli richiesti sudo apt install php php-cli php-fpm php-json php-common \ php-mysql php-zip php-gd php-mbstring php-curl \ php-xml php-pear php-bcmath

2. Installare Composer

php -r "copy('https://getcomposer.org/installer', 'composer-setup.php');" php composer-setup.php php -r "unlink('composer-setup.php');" sudo mv composer.phar /usr/local/bin/composer composer --version

3. Configurare Database

sudo mysql CREATE DATABASE eshop CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci; CREATE USER 'eshop_user'@'localhost' IDENTIFIED BY 'password'; GRANT ALL ON eshop.* TO 'eshop_user'@'localhost'; FLUSH PRIVILEGES; exit # Importare database mysql -u eshop_user -p eshop < database.sql

4. Configurare Progetto

sudo cp -r ESHOP /var/www/html/ cd /var/www/html/ sudo nano config.php // Modificare config.php: define('DB_HOST', 'localhost'); define('DB_USER', 'eshop_user'); define('DB_PASS', 'password'); define('DB_NAME', 'eshop');

5. Impostare Permessi

cd /var/www/html/ESHOP sudo mkdir -p uploads/{products,receipts} logs temp sudo chown -R www-data:www-data . sudo find . -type d -exec chmod 755 {} \; sudo find . -type f -exec chmod 644 {} \; sudo chmod -R 775 uploads logs temp

6. Accesso Pannello Admin

  • URL: http://localhost/admin/
  • Credenziali predefinite:
    • Username: admin
    • Password: admin123
IMPORTANTE: Cambiare immediatamente la password al primo accesso!

7. Configurare Apache

sudo a2enmod rewrite sudo systemctl restart apache2

Importante

  • Verificare che tutte le cartelle abbiano i permessi corretti
  • Testare il caricamento file in uploads/products e uploads/receipts
  • Controllare i log di PHP e Apache per eventuali errori
  • Modificare le password predefinite del database
